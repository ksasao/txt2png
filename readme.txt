■ これはなんですか？
csv形式(UTF-8)のファイルを読み込み、指定したサイズの長方形を
埋め尽くすようにレンダリングを行います。

■ 使用前の準備
- Windows XP 以前の場合
.NET Framework 3.5 SP1 をインストールしてください。

- Windows Vista 以降の場合
特にインストールする必要はないと思います。SP1を要求されたら
Windows Update を行ってください。

■ 動作環境
- 2GB以上のメモリ
- 1GHz以上の動作クロック
- 100MB以上の空きディスク領域

■ 使い方
1) data.csv というファイルを txt2png.exe と同じフォルダに置き
    ます。data.csv は1行目がヘッダ、2行目以降は、
    任意の文字列,ユーザー名(, メッセージ)
    の書式である必要があります。
2) txt2png.exe を起動します。あとは適当に使ってみてください。
    画像生成ボタンを押してしばらくすると、txt2png.exe と同じ
    フォルダにレンダリング結果が出力されます。

■ 既知の制限事項
出力サイズの最大値は 16383×16393 です。

■ ライセンス
NYSL (煮るなり焼くなり好きにしろライセンス) です。
